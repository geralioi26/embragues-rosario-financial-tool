# Financial Tool - Embragues Rosario ⚙️
**Status: Under Development / Estado: En desarrollo**

### [EN] Description
Comprehensive financial management tool under development for **Embragues Rosario**. This system centralizes the calculation of coefficients, platform commissions (Getnet, +Pagos), and local taxes (VAT), operating 100% in Argentine Pesos (ARS).
